## Definitions:

### Data Model: 
    Collection of concepts for describing data.
### Schema: 
    Description of a particular collection of data, using given data model.
    Specified during DB design.
    Not expected to changed frequently

### Relational Model of Data:
    Relation through table with columns and rows.

### Types of Schema:
    Physical Schema
    Conceptual Schema or Logical Schema  like a 3d model of a building High Level
    Multiple Views or External Schema
### Importance of DB
    Logical Data Independence
    Physical Data Independence
